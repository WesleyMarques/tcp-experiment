This directory contains the coverage information of used versions of the six subject projects, the "sorted_version.txt" file under the main directory of each project provides the sequence of all the versions from latest to oldest(top-down). In each version's directory, the "running_time.txt" file provides all the test cases in each line. The "statement_matrix.txt" provides the statement coverage infomation for each test case, the Nth line means the Nth test case in the "running_time.txt" covers which statements, '1' means covered and '0' means not covered.The "method_matrix.txt" and "branch_matrix.txt" provide the similar information except the coverage types are method and branch.
