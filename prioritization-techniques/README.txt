The three files are time-aware techniques: total, addtional, ILP-total, ILP-addtional.

1.the file "total_and_addtional.py", is total and addtional techniques. The input file is the coverage information boolean matrix of each test case.

2. the file "ILP_total.py" is the ILP total technique. The input file is the coverage informatoin boolean matrix and the running time information of each test case in Vn-1 and Vn.

3. the file "ILP_addtional.py" is the ILP addtional technique. The input file is the coverage informatoin boolean matrix and the running time information of each test case in Vn-1 and Vn.


2 and 3 should run in GUROBI environment.
